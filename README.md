# Etienne McDowell - Portfolio Website

Portfolio website showcasing graphic design, illustration, and typography work.

## Features

- **Work Gallery**: Browse projects with image transitions and lightbox functionality
- **Project Detail Pages**: Detailed views for each project with multiple images
- **Info Section**: Bio, contact information, and CV download
- **Responsive Design**: Optimized for desktop and mobile devices

## Projects

- SLAM Ceramics
- Lukx Merchandise
- Lakeside Wine Branding
- Lil'Tone Album Cover
- Element Skateboard
- Driftwire Brand Identity
- Bawette Magazine
- Calendar Summits of the World
- More Client Work and Various Projects

## Technologies

- HTML5
- CSS3
- JavaScript (Vanilla)
- No build process required - static site

## Setup

1. Clone the repository
2. Open `test-site-perso.html` in a browser
3. No build or installation required

## Deployment

This site is deployed on Vercel and automatically updates when changes are pushed to the main branch.
